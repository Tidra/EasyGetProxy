 Clash

配置文件
通用配置文件 {{ .scheme }}://{{ .domain }}/clash/config 一键导入
mihomo-mrs 规则组配置文件 {{ .scheme }}://{{ .domain }}/clash/config?f=mrs 一键导入
mihomo-geodata 规则组配置文件 {{ .scheme }}://{{ .domain }}/clash/config?f=mihomo 一键导入
通用配置文件(附带节点筛选样例) {{ .scheme }}://{{ .domain }}/clash/config?t=all 一键导入
以上提供的配置文件本身不包含节点，而使用clash provider的方式加载远程节点。
Provider
所有节点 {{ .scheme }}://{{ .domain }}/clash/proxies
筛选节点示例 {{ .scheme }}://{{ .domain }}/clash/proxies?c=CN,HK,TW&s=15,30&t=ss
❗️节点过多会造成软件卡顿，可以自行筛选节点后修改配置文件的proxy-provider部分。
Provider筛选参数

🌟 使用URL Query对Clash Provider进行筛选

Query Key Query Value 备注
类型 t http,https,socks5,ss,ssr,vless,vmess,trojan,hysteria,hysteria2,snell clash默认不含vless,hysteria,hysteria2
其他默认all代表全部类型
可同时选择多个类型
排除类型 nt http,https,socks5,ss,ssr,vless,vmess,trojan,hysteria,hysteria2,snell 可同时选择多个类型
国家 c AT,CN,IN,HK,JP,NL,RU,SG,TW,US... 可同时选择多个国家
排除国家 nc AT,CN,IN,HK,JP,NL,RU,SG,TW,US... 可同时选择多个国家
速度 s 任何数字 单位: mb/s
单个数字选择最低速度
两个数字选择速度区间
只需有效节点 v true,false 筛选有效或全部节点
